（1) 用户登录：用户填写用户名、密码完成登录
（2) 新增科室：名字长度至少俩位，可以新增科室
（3）新增医生：名字长度至少俩位，可以新增医生
（4）新增患者：名字长度至少俩位；手机号格式要求：11 位数字，以 1 开头（正则/^1[3-9]\d{9}$/，可以新增患者
（5）新增出诊：医生号必须为数字（type: ‘number’），出诊日期必填（通过日期选择器el-date-picker输入，格式为日期类型）总放号数和剩余放号数必须为大于 0 的数字（type: ‘number’, min: 1），可以新增出诊
（6）分页查询：科室管理根据科室名模糊查询，医生管理根据医生名和科室号模糊查询，出诊管理和预约管理根据科室名模糊查询，患者管理根据患者名模糊查询，可通过选页跳转
（7）页面无刷新切换：用户点击不同页面可不刷新切换
（8）按医生出诊表的剩余放号数来设定是否允许预约
前端：Vue3
后端：SpringBoot
数据库：MySQL
